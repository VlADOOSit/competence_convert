Програмна система складається з двох сервісів ml_service та api

Всі клієнтські запити будуть оброблятись через api сервіс (в локальному випадку це http://localhost:3001/api/...)

Для запуску рекомендується використання Docker.

Для запуску достатньо клонувати репозиторій та виконати 
```
docker compose up -d --build
```

!!! Перший запуск буде достатньо довгий так як модель досить велика

Для зупинки 
```
docker compose down
```

Для того щоб конвертувати вакансію у компетенції використовується даний ендпоінт
```
http://localhost:3001/api/prediction/run
```
Як тіло запиту вказати form-data назва file тип file (тип файли html) приклад файлe в ./api-service/test.html. Приклад запиту на скріншоті

Перший запит також потребує деякого часу так чк модель підвантажується не одразу я додав axios-retry. Всі наступні запити відпрацьовують добре
